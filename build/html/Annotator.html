Annotator

Other than managing the mapping projects, the admin user is also responsible for managing the users that will be participating in the projects, ie. the Annotators. To support this, the following tools are provided under the Annotator tab.

  • Add Annotator

  • List of Annotator

    • Disable Annotator

    • Disabled Annotators

    • Change Annotator Role

    • Show or Change Annotator Tags

  • Manage Annotator Tags

Add Annotator 

To allow a new user to start participating in the mapping annotation projects, the admin user can use the button to manually add an existing platform user to the Vector Annotator Application by specifying their email and annotator role.

Annotator Requests 

The Admin user can use the button to obtain a list of historical requests that users have submitted to register as an Annotator through the tools:Annotator Role Registration tool. That includes the approved, rejected, and pending requests. The Admin user will be able to approve or reject a pending request.

List of Annotator

A list of the current Annotators are listed under the Annotator Management tab, with their Name, Email, and Role. The Admin user can filter on search for Annotators and also use the to view more details of an Annotator.

Other than viewing information about Annotators, the following actions can be performed on each Annotator:

  • Disable Annotator

  • Change Annotator Role

  • Show or Change Annotator Tags

Disable Annotator 

Clicking on the button disables the target Annotator. The Annotator will not be able to access mapping entry nor any of the mapping projects. The Admin user can use the Disabled Annotators tool to see a complete list of disabled Annotators and re-enable them if needed.

Disabled Annotators 

The Admin user can click the button to bring up a list of Annotators that had been disabled. They can also use the button to re-enable an Annotator if they wish.

Change Annotator Role 

The role of an Annotator can be changed at any time by the Admin user using the button.

Show or Change Annotator Tags 

Annotator tags can be created for easier access management. The Admin user can also view or chnage the Annotation tag of any user using the tool.

Manage Annotator Tags 

All the Annotator Tags are displayed and can be managed with the tool.

  • Use to add a new tag.

  • Click on the name of an existing tag for more actions:

    • Use to delete the selected tag.

    • Use to rename the selected tag.

    • Use to display a table of all Annotators with the selected tag on the right panel.

    • Use to hide the table of tagged Annotators on the right panel.

  • When a table of tagged Annotators is displayed on the right panel:

    • Use to add one or multiple Annotators under the selected tag.

    • Use the checkboxes to select one or multiple Annotators and click to remove the selected tag for them.
