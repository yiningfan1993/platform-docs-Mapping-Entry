Interface Walkthrough

Once users entered the Workflow Assembler Application, they will see the interface below, which contains the following main functions.

Sort

Sort function allows users to sort the items in the workflow element list based on the following criteria:

  1. Alphabetic : sort item list based on alphabetic order. This is the default order for all lists.

  2. Created Time : sort item list based on time the items are created in descending order.

Item Types Dropdown

Users can switch the list among three workflow items:

  1. Gallery

  2. Workflow/Space

  3. Worker

Users can also use the search function below the dropdown to quickly locate the target item in the list.

Create New Item

By clicking on the Create button, users can create a new workflow element with other customizations.

To create a new gallery, the following information needs to be provided:

  1. Type: Gallery

  2. DAG Class: Select between 2D and 3D

  3. Name: Name for the newly created Gallery. Can only contain letters, numbers and “_”.

  4. Advanced Config (optional): Set advanced configuration for this gallery if necessary.


To create a new workflow/space, the following information needs to be provided:

  1. Type: Workflow/Space

  2. Sub Type: Select between workflow and space

  3. Name: Name for the workflow/space. Can only contain letters, numbers and “_”.

  4. Workflow Type: Specify the workflow type (new workflow only)

  5. Advanced Configuration (optional): Set advanced configuration for this gallery if necessary.

Item List

All existing items will be listed in the item list. The list can be swtiched based on different types of items in the Item Types Dropdown.

  1. By clicking on the title of the items in the list, the item details will be displayed in the DAG Graph Area.

  2. By hovering the cursor over the title of the item in the list, users can click to copy the name of the item.

  3. By clicking on the button next to the item title, the item can be deleted from the list.

DAG Graph Area

DAG graph area depicts the workflow of selected item. Users can also customize the working item by adding or re-directing the process flow in the graph area.

View-Only Mode

In view-only mode, users can view details of each node of the workflow by double-clicking on the dot that represents the node in the DAG graph.

Users can go back to the previous workflow by clicking on the name in the hierarchy.

To view more details of the workflow, the following actions are available:

  1. Dragging Node: Users can hold-click to drag the nodes inside of the DAG graph to gain better view of the workflow.

  2. Panning Graph: Users can pan to different area of the graph by hold-click on the empty space of the graph.

  3. Zoom In/Out: Users can zoom in/out in the graph area by scrolling the mouse wheel.

Edit Mode

By clicking on the button on the top-right corner of the window, the edit mode is activated.

Under edit mode, users can perform following actions in the DAG graph.

Add/Remove Node

By clicking the + button on top of the DAG graph area, users can add new node into the DAG graph.

In the prompt window, users can either filter the selections using the Filter Tags or directly type the node name in the dropdown below to quickly locate the target note.

By clicking OK button, a new node will be added to the graph without any connection to the existing nodes.

After clicking on the target node and it is highlighted in blue, the selected node can be removed by clicking -.

Create/Remove Connection Between Nodes

There is no connection from or to the new node that is added to the graph. To create a connection to other node in the graph:

  1. Single-click on the starting node to highlight it.

  2. Double-click on the destination nodes, note that an arrow appears between the starting node to destination node, which means the connection is established.

To remove am existing connection,

  1. Single-click on the connection line to highlight the connection.

  2. Click on -, same as one that removes node, on top of the DAG graph area to remove the connection.

  3. In the prompt window, click OK to confirm the action. Notice that the connection between two nodes is removed.

JSON Detail for Selected Item

This panel provides detailed description for selected item in JSON format.

Users can:

  1. Navigate and expand the JSON file to see more details;

  2. Use copy to clipboard function to copy the whole or part of the JSON snippet out.

Advanced Configuration for Selected Item

Users can view adavanced configuration for the selected items if it is available.

Add or Remove Items in Graph

After clicking button on the top-right corner of the application to activate the editing mode

Actions: Edit, Save, Lock, Copy and Cancel Edit

After users open an item from the list panel, the following tools are activated for users to modify the selected item.

  1. Edit: By clicking on the button, the editing mode is activated and users can edit the links/nodes in the DAG graph. Users can also cancel current editings by clicking on button.

  2. Save: By clicking on the button, all edits will be saved to the selected item that the user is currently working on.

  3. Lock: By clicking on the button, users can lock current item and the current item cannot be edited unless it is unlocked by clicking button.

  4. Copy: By clicking on the button, the currently selected item will be duplicated. By default, the duplicated item will have “_cp” in its name.

  5. Full-screen: By clicking on the button, the DAG graph, JSON details and advanced configuration panels will enter full screen mode. Users can exit full-screen mode by clicking .

Current Breadcrumb Navigation

The breadcrumb navigation section indicates the hierarchy among gallery, workflow/space and worker. Users can navigate from gallery to worker or view any items in the hierarchy by clicking on the titles.
